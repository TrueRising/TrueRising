# Hero media assets

Drop your production hero video files here before deployment:

- `obsidian-hero.webm` (preferred, compressed high-quality web delivery)
- `obsidian-hero.mp4` (fallback)
- `obsidian-hero-poster.svg` (already included as placeholder poster)

Recommended export targets:
- 1920x1080 or 2560x1440
- H.264 for MP4 fallback
- VP9/AV1 for WebM
- 6-12 seconds seamless loop, muted
- < 6MB per file for fast edge delivery
